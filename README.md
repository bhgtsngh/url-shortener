#  URL Shortener

A minimal URL shortener built using **Python** and **Flask**.

##  Features
- Shorten long URLs
- Redirect using short codes
- Track click counts
- View basic metadata

##  Run the App
```bash
python -m flask --app app.main run

Run Tests
>> pytest tests
Structure
> app/: Core logic and routes
> tests/: Pytest test cases
```
